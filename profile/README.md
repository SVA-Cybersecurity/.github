# Über uns | About Us
🇩🇪 Die SVA System Vertrieb Alexander GmbH ist ein deutscher IT-Dienstleister. Unter anderem unterstützen wir Kunden in allen Bereichen der Cyber Security – von GRC über Penetration Testing und Red Teaming bis hin zu Incident Response und Security Awareness.

Dieses GitHub-Profil sammelt Tools, Skripte und Ressourcen, die bei unserer Arbeit entstanden sind und die wir der Community zur Verfügung stellen.

🇬🇧 SVA System Vertrieb Alexander GmbH is a German IT service provider. Among other things, we support our customers in all areas of cyber security – from GRC and penetration testing to red teaming, incident response, and security awareness.

This GitHub profile collects tools, scripts, and resources that have come out of our work and that we make available to the community.
